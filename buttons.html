Enchant buttons

Enchant offers five different buttons in a lot of different styles. Buttons are available in a box or pill shape and in a solid or an outlined form. All buttons have similar effects on hover and on click.

Solid buttons

These are the different solid buttons that are available.

Standard

Source code:

<button class="btn">Standard</button>

Primary

Source code:

<button class="btn btn-prim">Primary</button>

Secondary

Source code:

<button class="btn btn-sec">Secondary</button>

Danger

Source code:

<button class="btn btn-danger">Danger</button>

Warning

Source code:

<button class="btn btn-warning">Warning</button>

Outlined buttons

These are the different outlined buttons that are available.

Standard

Source code:

<button class="btn-outline">Standard</button>

Primary

Source code:

<button class="btn-outline btn-prim">Primary</button>

Secondary

Source code:

<button class="btn-outline btn-sec">Secondary</button>

Danger

Source code:

<button class="btn-outline btn-danger">Danger</button>

Warning

Source code:

<button class="btn-outline btn-warning">Warning</button>

Buttons

Enchant offers five different buttons in a lot of different styles. Buttons are available in a box or pill shape and in a solid or an outlined form.

Pill-shaped buttons

These are the pill-shaped buttons. The pill shape is also available for outlined buttons. All you need to do is add the btn-pill class.

Standard

Source code:

<button class="btn btn-pill">Standard</button>

Primary

Source code:

<button class="btn btn-pill btn-prim">Primary</button>

Secondary

Source code:

<button class="btn btn-pill btn-sec">Secondary</button>

Danger

Source code:

<button class="btn btn-pill btn-danger">Danger</button>

Warning

Source code:

<button class="btn btn-pill btn-warning">Warning</button>

Disabled buttons

To disable a button, simply add the disable class. This works for all buttons.

Disabled

Source code:

<button class="disable btn btn-pill">Standard</button>
